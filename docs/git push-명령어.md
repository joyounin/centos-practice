git 기본명령어
```
echo "# mariadb-practice" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/joyounin/mariadb-practice.git
git push -u origin main
```
git 설정 뒤 올리는 법
```
git add -A                    # git 모든파일 추가
git commit -m 'commit'        # git commit message 입력
git push                      # git push
```
